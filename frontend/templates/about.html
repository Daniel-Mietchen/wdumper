How it works

This tool provides a simple interface to create customized RDF dumps from Wikidata. It is based on Wikidata Toolkit.

The dumping process logically consists of two steps. First, the entities to export are determined. Those entities are then exported according to the rules. This step uses the rules for statements to determine what to export for each statement of the selected entity. In addition to the statements, you can also control what other information is exported for the seletced entity, such as labels.

How to contribute

The source code for the application is hosted on GitHub. Feel free to create issues for any bugs you discover or features you like to see implemented. Pull requests are of course also appreciated!
